# Leather Mold Generator

Leather Mold Generator is a Blender add-on for creating leather-forming molds from STL models.

## Features

- Male mold generation
- Female mold generation
- Leather thickness compensation
- Draft angles
- Alignment pins
- Vent grooves
- STL export

## Requirements

- Blender 5.1 or newer
